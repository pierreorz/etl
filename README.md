## 使用 🐂

1. 安装依赖
```shell
pnpm install
```

2. 开发
```shell
pnpm run dev 
```

3. 预览
```shell
pnpm run preview
```

4. 打包

```shell
pnpm run build
```

5. 单元测试
```shell
pnpm run test
```

6. 单元测试报告生成
```shell
pnpm run coverage
```

7. 样式报告预览

```shell
pnpm run analysis
```

8. 样式报告打包

```shell
pnpm run analysis:build
```

9. 类型检查

```shell
pnpm run typecheck
```
